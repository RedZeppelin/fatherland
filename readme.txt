Fatherland Beta 1.0
A post-ww2 scenario with Germany victorious.

Compatibility:
- Darkest Hour Full 1.03 RC3  version, unmodified.

Installation instructions:
1: Copy a clean install and unmodified version of Darkest Hour Full 1.03 RC3 and rename it "Fatherland".
2: Download the mod.
3: Copy contents of the downloaded archive into the newly copied directory (agree to replace all the files when asked).
